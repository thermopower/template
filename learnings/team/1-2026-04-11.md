## 적용한 개선

### 1. `.ruler/AGENTS.md` — `Dependency Injection Rules (FastAPI)` 섹션 신규 추가
- Application 레이어에서 Infrastructure 구현체 직접 import/생성 금지 명시
- FastAPI `Depends(factory_func)` 패턴 강제, 직접 인스턴스화 금지
- 폴백 구현체 선택 로직은 팩토리 함수에서 처리하도록 명시
- `httpx.AsyncClient` 등 연결 풀은 `lifespan` 또는 `Depends` 스코프에서 생성·해제 의무화
- 테스트 교체는 `app.dependency_overrides[factory_func]` 사용 명시

### 2. `.ruler/AGENTS.md` — `Coding Rules` 에러 처리 항목 강화
- `except` 블록에서 빈 값 반환 시 `logger.warning` 이상 로그 필수
- silent pass(`except: pass`, `except: return {}`) 명시적 금지
- 허용/금지 예시 코드 추가

### 3. `.ruler/AGENTS.md` — `Coding Rules` 공유 상수 단일 정의 원칙 추가
- 2개 이상 모듈이 참조하는 상수는 단일 정의 원칙 적용
- Domain 상수는 `src/domain/`에 먼저 정의
- private 심볼(`_` 접두어) 레이어 경계 넘기기 금지

### 4. `.ruler/AGENTS.md` — `테스트 격리 규칙 (FastAPI / pytest)` 섹션 신규 추가
