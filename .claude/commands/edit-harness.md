# /edit-harness

> **참고**: 이 저장소의 기본 모드는 이미 일반 편집 모드입니다. 세션 시작 시 하네스 루프는 자동으로 실행되지 않습니다.

하네스 루프가 활성화된 상태(`/harness` 실행 후)에서 일반 편집 모드로 돌아옵니다.

## 실행 후 동작 규칙

1. **하네스 루프 비활성화**: requirement-writer, planner, sprint-builder 등 하네스 에이전트를 자동으로 실행하지 않는다.
2. **단계 전환 규칙 무시**: CLAUDE.md 섹션 2의 단계 전환 규칙을 적용하지 않는다.
3. **일반 편집 모드**: 사용자의 요청을 직접 수행한다.

## 하네스 루프로 돌아가려면

`/harness` 를 실행한다.
