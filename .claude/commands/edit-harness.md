# /edit-harness

하네스 템플릿 수정 모드로 전환합니다.
이 커맨드를 실행하면 하네스 루프(requirement-writer → planner → sprint-builder → ...)를 실행하지 않습니다.

## 이 커맨드가 필요한 이유

이 저장소는 하네스 템플릿 자체입니다. 평소에는 CLAUDE.md의 하네스 운영 규칙이 활성화되어 있어,
무언가를 말하면 requirement-writer 실행을 시도합니다.
템플릿 자체를 수정할 때는 이 커맨드로 하네스 루프를 비활성화합니다.

## 실행 후 동작 규칙

이 커맨드가 실행되면 현재 세션에서 다음 규칙을 따른다:

1. **하네스 루프 비활성화**: requirement-writer, planner, sprint-builder 등 하네스 에이전트를 자동으로 실행하지 않는다.
2. **단계 전환 규칙 무시**: CLAUDE.md 섹션 2의 단계 전환 규칙을 적용하지 않는다.
3. **일반 편집 모드**: 사용자의 요청을 직접 수행한다. 파일 읽기, 수정, 에이전트 파일 편집 등을 자유롭게 진행한다.
4. **Superpowers 스킬 억제 유지**: 스킬 억제 정책(섹션 0)은 편집 모드에서도 유지된다.

## 종료

세션을 재시작하면 하네스 모드로 돌아온다.
현재 세션에서 하네스 모드로 돌아오려면 `/exit-edit-harness`를 실행한다.
