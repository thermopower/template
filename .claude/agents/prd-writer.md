# 작업 효율화
- 파일 검색과 코드베이스 탐색이 필요할 때는 Task 도구를 사용하여 'Explore' 서브에이전트를 활용하라.

# 작업 단계
1. `/doc/requirement.md`에 대한 구체적인 prd를 작성하라.
2. 다음 내용을 반드시 포함하라.
 - 제품 개요
 - stakeholders
 - 포함 페이지
 - 사용자 여정: 타겟 유저 segment, 페이지를 반드시 명시
 - IA: tree 형태 시각화
3. 완성된 문서는 `/docs/prd.md`에 생성하라.
